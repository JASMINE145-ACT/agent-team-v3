# 万鼎选型知识库（面向 Agent 选型）

本文件用于指导 `match_quotation` / `match_quotation_batch` 在多候选时的选择。  
目标：减少“选错品类、选错冷热、选错规格、过度依赖来源”的问题。

---

## 1) 选型总原则（硬约束）

按下面顺序决策，前一层不满足时不得进入后一层：

1. **品类语义一致性（最高优先）**
2. **关键规格一致性（必须）**
3. **次级属性比较（压力等级/S 系列/品牌）**
4. **来源加权（仅作 tie-breaker，不是一票否决）**

### 1.1 来源规则（修订后）

- `source="共同"`（历史+字段同时命中）可信度通常更高；
- 但来源只在“语义与规格都接近”时生效；
- 当高来源候选与用户语义冲突时（如用户要“热水管”，候选是“冷给水”），**必须降权或排除**；
- 最终来源优先级：`共同 > 历史报价 > 字段匹配`（仅在前 1-3 层均接近时使用）。

### 1.2 典型纠偏（热水管案例）

用户：`PPR热水管3/4寸`  
如果候选中“字段匹配”有 `热给水直管 dn25 (3/4")`，而“历史报价”是 `冷给水直管 dn25 (3/4")`，应优先热给水直管。  
**结论：语义一致性（热/冷）高于来源优先级。**

## 2) 同义词与外语标准化（与代码匹配逻辑保持一致）

### 2.1 同义词组（任一出现即等价）

- 直接 / 直接头 / 直通 / 直通接头
- 变径 / 异径
- 大小头 / 异径直通 / 异径套 / 变径直接 / 异径直接
- 内丝 / 内螺纹；外丝 / 外螺纹
- 锁母 / 锁扣 / 管接头
- 穿线管 / 电线管
- 承插 / 承插式
- 堵头 / 管帽

### 2.2 弯头角度（强约束）

- 半弯 => 45°弯头（不得与“弯头”泛化混同）
- 全弯 / 直角弯 => 90°弯头
- 未写“半弯”的“弯头”默认按 90°优先

### 2.3 外语/口语映射

- `4 cabang` => 管四通圆接线盒
- `conduit` / `counduit` => 电线管
- `pipa` => 管
- `socket` / 套筒 => 管直通
- `klem` => 管夹
- `cabang` / `tdust` => 四通
- 热熔器 / 热熔机 / 熔接器 => 焊接机
- 四通接线盒 => 管四通圆接线盒
- 马鞍卡 => 管夹（电工套管场景通常指 U 型管夹）

## 3) 规格规则（必须命中）

## 3.1 中文/英制尺寸到 DN 映射

- 2寸 => DN50 (2")
- 1寸 => DN25 (1")
- 6英寸 => DN150 (6")
- 1½寸 / 1.5寸 => DN40 (1-1/2")
- 3/4寸 => DN20 (3/4")
- 1½ in / 1-1/2 in => DN40 (1-1/2")
- 3/4 in => DN20 (3/4")

### 3.2 主径×副径（A*B / A×B / AB）规则

- 形如 `32*20` / `32×20` / `3220`：A 为主径（dn），B 为副径（英寸侧）
- B 与英寸映射：
  - 15/16/20 => 1/2"
  - 25 => 3/4"
  - 32 => 1"
  - 40 => 1-1/4" 或 1.25"
  - 50 => 1-1/2" 或 1.5"
- A 与 dn 映射：20=>dn20，25=>dn25，32=>dn32，40=>dn40
- 示例：`32*20内丝三通` => 优先 `dn32x1/2"`，不选 `dn32x3/4"`
- 若仅单数值（如 `32弯头`）且无副径，不推断副径

## 4) 场景型业务规则

1. 三角阀 != 角阀；仅有角阀候选时判无匹配  
2. 软管：库内无软管类产品，询价含“软管”返回无匹配  
3. 规格精确匹配优先：dn25 不选 dn20  
4. 排水场景优先 PVC-U 排水配件，不选 PPR 给水系列  
5. “50直接/110直接”通常是排水直通(管箍)，优先 PVC-U 排水配件  
6. “50cm”表示长度，不得误判为 dn50 管径  
7. 等径三通优先于异径三通（除非用户明确要异径）  
8. 同等条件下联塑品牌优先  
9. `90°弯头 + 110`：优先 PVC-U 排水 90°弯头，不选 PPR/PE 配件  
10. 电工套管场景：
   - conduit / 电线管 => PVC 电线管(B管)
   - socket => 管直通(套筒)
   - klem / 马鞍卡 => 管夹(U型)
   - tdust 4 cabang => 管四通圆接线盒(带盖)
   - 热熔器 => 焊接机
